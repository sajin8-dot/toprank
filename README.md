# TopRank - Bullet Journal Learning Tracker

A responsive, kid-friendly Progressive Web App (PWA) to track kids' daily school learning with a visual Bullet Journal style.

## Key Features

* **Multi-Kid Dashboards**: Toggle between boards for **Rahel**, **Elsa**, and **Eliah**.
* **Subject Overview Accordion Dashboard**: See preparation progress across subjects, automatically sorted with the weakest subject on top.
* **Memory Decay Simulation**: Subsection ratings decay linear-style by `-0.5` points per day, motivating timely review.
* **Drag-to-Update Sliders**: Easy touch-friendly slider inputs to log new proficiency ratings (0-10 scale).
* **Quizzes Countdown**: Add upcoming quizzes and view days remaining.
* **Activity Log**: Event history logging with emojis.
* **Installable PWA**: Works offline and can be installed as a standalone app on Android.
